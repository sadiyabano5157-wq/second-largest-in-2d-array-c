# second-largest-in-2d-array-c
A simple C program that finds the largest and second largest elements in a 2D array using nested loops.

# Largest and Second Largest Element in 2D Array (C Program)

## 📌 Description
This program finds the largest and second largest elements in a 3×3 matrix.

## 🚀 How It Works
- Traverse all elements of the matrix using nested loops.
- Track the largest element.
- Track the second largest element.
- Display both values.

## 📥 Input Matrix

```text
12 45 7
89 23 56
11 90 34
```

## 📤 Output

```text
Largest Number = 90
Second Largest Number = 89
```

## 🛠️ Concepts Used
- 2D Arrays
- Nested Loops
- Conditional Statements
- Largest and Second Largest Logic

## 🎯 Learning Outcome
- Matrix traversal
- Comparing values
- Tracking multiple maximum elements
- Problem solving with arrays
